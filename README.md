# SeleniumAutomationFW

Selenium, Java, testNG automation framework and Allure reports.

Requirements :

To run this project you will need to have installed in your machine:

- JDK 1.7 or higher
- Maven 3.0 or higer

Maven Execution:

"mvn clean test -Dsuite= TESTNG_SUITE_TO_RUN"
